# Selection-Style Archiv

Reine Archiv-Kopien der bisherigen Auswahl-UIs für Theme / Wallpaper / Screenshots / Videos.
**Nicht in `shell.qml` eingebunden** — liegen hier nur als Referenz/Backup, werden nicht geladen.

## Hearthstone (Card-Deck)
Gefächerte Karten, „Felt"-Hintergrund. Deckt alle 4 Funktionen ab.
- `Hearthstone-ImageCarouselPanel.qml` → Theme + Wallpaper
- `Hearthstone-MediaBrowserPanel.qml`  → Screenshots + Videos (Video-Poster via ffmpegthumbnailer)

## Carousel (Skewed Slices)
Schräg gestellte Streifen, eines expandiert in der Mitte. Kam dem Bar-Stil näher,
aber noch nicht final. Nur Theme/Wallpaper (entstand vor dem MediaBrowser).
- `Carousel-ImageCarouselPanel.qml` → Theme + Wallpaper

## Status
Beide Stile passen noch nicht 100% zum minimalistischen Bar-Look (Paper/Ink/Seal,
dünne 1px-Pillen, JetBrainsMono). V1 bekommt eine neue, minimalistische Auswahl-UI;
diese hier bleiben als Fallback erhalten.
